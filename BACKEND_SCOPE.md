## Seed Data

Two seed scripts are available:

| Script | Purpose |
|---|---|
| `seed_daraz_mvp.sql` | Minimal: one brand, one product, one seller, one marketplace. Use for fastest local boot. |
| `seed_all.sql` | Full: 5 marketplaces, 3 brands, multiple products, listings, violations, enforcement letters. Use for demo. |

### Fresh boot sequence (minimal)

```powershell
# 1. Start services
docker compose up -d mysql redis

# 2. Load schema
Get-Content backend\database\schema.sql | docker exec -i mysql_db mysql -uroot -p<password>

# 3. Load seed
Get-Content backend\database\seed_daraz_mvp.sql | docker exec -i mysql_db mysql -uroot -p<password> verifishelf

# 4. Start backend
python -m uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```
